Template maintenance guide

Recommended future workflow: update the data only, not the layout.

1. Open the JSON block near the bottom of this file: <script id="report-data" type="application/json">.

2. Change the report period, counts, tables, and daily-grid arrays.

3. Keep the structure the same unless you intentionally want a new section.

The page will rebuild itself from that JSON when opened in any browser. No external assets or libraries are required.
